    **Step 1 -- YAML template:**
    Rewrite `templates/orca_template.yaml` into the machine-readable format from the RESEARCH.md schema. The YAML MUST produce data matching the hardcoded constants in spectrace_annotator.py lines 43-148 EXACTLY -- same layer names, same order, same parent groups, same colors (hex equivalents of the RGB tuples), same UI sections.     **Step 2 -- templates.py implementation:**
    Create `spectrace/core/templates.py` implementing:

    1. `TemplateError(Exception)` -- raised for all template issues.

    2. `SpeciesTemplate` class with attributes: name, version, root_group_name, layer_structure (list of (name, parent, type) tuples), layer_colors (dict of path-qualified name to (r,g,b) tuple), layer_sections (list of (section_name, [layer_paths])), spectrogram_params (dict). Methods: `get_layer_names()` returns set of path-qualified layer names (excluding groups).

    3. `hex_to_rgb(hex_str)` -- converts '#RRGGBB' to (R, G, B) tuple. Raises TemplateError on invalid input.

    4. `load_template(template_name='orca', templates_dir=None)` -- loads `{template_name}_template.yaml` from templates_dir (default: `templates/` relative to project root). Calls `_validate_and_build()`. Raises TemplateError if file missing or malformed.

    5. `_validate_and_build(raw, template_name, path)` -- validates required top-level keys ('template', 'root_group', 'layers', 'ui_sections'), parses layer hierarchy via `_parse_layers()`, validates UI section layer references exist, builds and returns SpeciesTemplate.

    6. `_parse_layers(items, parent_name, structure, colors)` -- recursively walks YAML list. For 'group' items: appends (name, parent, 'group') and recurses into children. For 'layer' items: appends (name, parent, 'layer'), builds path-qualified name, validates color exists, converts hex to RGB. Raises TemplateError on unknown item types.

    7. `discover_templates(templates_dir=None)` -- lists available template names by scanning for *_template.yaml files.

    The default templates_dir calculation: `os.path.join(os.path.dirname(os.path.dirname(os.path.dirname(os.path.abspath(__file__)))), 'templates')` -- this navigates from spectrace/core/templates.py up three levels to project root, then into templates/.

    Follow the code examples in RESEARCH.md Pattern 1 and Pattern 3 closely but ensure all formatting uses % operator, not f-strings.
